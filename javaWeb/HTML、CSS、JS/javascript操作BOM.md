## 脚注

1. [^银行家算法]:**1.四舍**：保留位数的后一位数字 **≤4** 时，直接舍去。**2. 六入**：保留位数的后一位数字 **≥6** 时，进位。**3. 五留双（关键规则）**： 当保留位数的后一位是 **5** 时，根据前一位数字的奇偶性决定：**前一位为奇数**：进位。**前一位为偶数**：直接舍去。**好处：减少累积误差**：传统“五入”会偏向高估，而银行家算法通过“五留双”使舍入概率均等，避免系统性偏差。
